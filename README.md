# Retail-Sales-Performance-Dashboard
An interactive Power BI dashboard designed to analyze retail sales performance, highlighting top and bottom products, discount effectiveness, and city-wise sales distribution to support data-driven decision-making.

🎯 Project Objective
The objective of this project is to analyze retail sales data to identify top-performing and underperforming products, evaluate discount effectiveness, and understand regional sales distribution to support data-driven business decisions.

📊 Dataset Used
1. Retail Sales Dataset
2. Fields included:
  * Product Name
  * Sales (Net Sale)
  * Profit
  * Quantity Sold
  * Discount Value
  * Promotion Name
  * City

❓ Key Business Questions
1. What are the Top 5 and Bottom 5 products based on sales?
2. Which promotions provide the highest average discount value?
3. How are sales distributed across different cities?
4. What is the overall total sales performance?

⚙️ Process Followed
1. Data Cleaning (removed null values, corrected formats)
2. Data Transformation using Power BI
3. Created measures (Total Sales, Average Discount, etc.)
4. Data Modeling (relationships between tables)
5. Built interactive visuals:
   * Bar Charts
   * Map Visualization
   * KPI Card
6. Dashboard Design & Formatting

📈 Dashboard Features
1. Top 5 & Bottom 5 Products by Sales
2. Average Discount Analysis by Promotion
3. City-wise Sales Distribution (Map View)
4. Total Sales KPI (129.47M)

💡 Project Insights
1. Apple iPhone 14 is the highest-selling product (~22M sales)
2. Weekend Flash Sale gives the highest discounts
3. Some products like Tupperware Lunch Box have very low sales
4. Sales are concentrated in major cities, indicating urban demand dominance
5. Discount-heavy promotions drive more engagement

✅ Final Conclusion

This dashboard provides a clear overview of product performance and sales trends. Businesses can use these insights to:

1. Focus on high-performing products
2. Improve low-performing product strategies
3. Optimize promotional campaigns
4. Target high-sales regions for growth
